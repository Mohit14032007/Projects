This project is a Java-based ERP (Enterprise Resource Planning) system that manages academic and administrative data. It includes features such as user management, data storage using databases, and application-level workflows. 
The project contains complete Java source code packaged as a ZIP file, two README/documentation PDF files explaining the system design and usage, and a demonstration video showcasing the working of the application.
