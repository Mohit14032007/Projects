RISC-V Assembler and Simulator (Python)
Overview

This project implements a simplified RISC-V assembler and simulator using Python.
The assembler converts RISC-V assembly instructions into 32-bit binary machine code, and the simulator executes the generated binary instructions while updating registers, program counter (PC), and data memory.

The project demonstrates core concepts of computer organization, instruction formats, and instruction execution flow.

Components
1) Assembler (assembler.py)

Converts RISC-V assembly code into binary machine code

Supports labels and branch offsets

Handles multiple instruction formats:

R-Type

I-Type

S-Type

B-Type

J-Type

Outputs a binary file compatible with the simulator

2️ Simulator (simulator.py)

Executes the binary instructions generated by the assembler

Maintains:

Program Counter (PC)

Register file (32 registers)

Data memory (predefined memory addresses)

Produces output showing:

PC value

Register values after each instruction

Final data memory state

Supported Instructions
Arithmetic & Logical

add, sub, slt, srl, or, and

Immediate & Memory

addi, lw, sw

Control Flow

beq, bne, jal, jalr

System Instructions

rst

halt

Register Set

The simulator supports standard RISC-V registers:

zero, ra, sp, gp, tp,
t0–t6, s0–s11, a0–a7

Data Memory

Predefined memory addresses from:

0x00010000 to 0x0001007C


Word-aligned memory access

Initialized with zero values

How to Run
Step 1: Assemble the program
python assembler.py input.asm output.bin

Step 2: Simulate execution
python simulator.py output.bin result.txt

Output Format

The simulator output file contains:

PC value after each instruction

Register values in binary

Final data memory contents

Project Highlights

Manual implementation of instruction encoding and decoding

Two’s complement handling for immediates

Label resolution for branch and jump instructions

Clear separation between assembling and execution stages

Technologies Used

Python 3

File I/O

Bit manipulation
